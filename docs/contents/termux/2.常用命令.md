## 2、安装ubuntu

### 2.1 先安装基础件proot-distro

```bash
 pkg install proot-distro 
```

### 2.2 查看可安装的Linux系统

```bash
 proot-distro list
```

### 2.3 安装Ubuntu

```bash
  proot-distro install ubuntu
```

### 2.4 安装完成后，进入 Linux(Ubuntu)环境

```bash
proot-distro login ubuntu
```
